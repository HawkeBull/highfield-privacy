# Highfield Privacy Policy

Last Updated: June 2026

Highfield is a private Discord bot used exclusively within a single Discord server.

## Information We Collect

* Message content only when required to process bot commands or configured features
* Server and channel IDs required for operation

## How We Use Information

The collected information is used solely to:

* Deliver messages from the Roblox game to Discord.
* Operate and maintain the bot's functionality.

## Data Sharing

We do not sell, rent, or share user data with third parties.

## Data Retention

Data is retained only for as long as necessary to operate the bot and may be deleted when no longer required.

## Contact

If you have questions about this Privacy Policy, please contact the server administrators.
