# Highfield Privacy Policy

**Last Updated:** June 2026

Highfield is a private Discord bot used exclusively within a single Discord server. It integrates with a Roblox game to synchronize messages, scheduled events, and other server activity between Discord and Roblox. Highfield is not available for public use.

## Information We Process

To provide its functionality, Highfield may process:

* Discord message content, including message edits and deletions, when required for synchronization with the Roblox game.
* Discord message metadata, such as message IDs, channel IDs, timestamps, and author IDs.
* Discord server IDs and channel IDs required for operation.
* Discord scheduled event information, including event names, descriptions, start/end times, and statuses.

Highfield only processes information necessary to perform its intended functions.

## How We Use Information

The information processed by Highfield is used solely to:

* Send messages and embeds from the Roblox game to Discord.
* Synchronize Discord messages, edits, and deletions with the Roblox game.
* Synchronize Discord scheduled events with the Roblox game.
* Operate, maintain, and improve the bot's functionality.

## Data Sharing

Highfield does not sell, rent, or share user data with third parties. Information processed by the bot is used only to provide its services between Discord and the associated Roblox game.

## Data Retention

Message content is processed only as needed to provide synchronization features and is not persistently stored.

Certain operational data, such as active Discord Scheduled Events, may be temporarily cached in the bot's memory while it is running to support synchronization with the associated Roblox game. This cached data is not shared with third parties, is never written to persistent storage, and is automatically discarded when the bot stops or restarts.

## Security

Reasonable measures are taken to protect information processed by Highfield from unauthorized access, disclosure, or misuse.

## Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected by updating the **Last Updated** date at the top of this document.

## Contact

If you have any questions about this Privacy Policy, please contact the administrators of the Discord server where Highfield is used.
